SISTEMA COM CADASTRO E LOGIN - PHP + SQLITE

COMO USAR:

1. Extraia a pasta sistema-login-sqlite para:
   C:\xampp\htdocs\

2. Ligue o Apache no XAMPP.

3. Acesse no navegador:
   http://localhost/sistema-login-sqlite

4. O banco de dados será criado automaticamente com o nome:
   banco.sqlite

ARQUIVOS DO PROJETO:

- index.php
- db.php
- cadastro.php
- login.php
- area_restrita.php
- logout.php

TECNOLOGIAS:

- PHP
- SQLite 3
- PDO
- HTML
- CSS
- Sessões PHP
- password_hash
- password_verify
